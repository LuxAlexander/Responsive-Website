This is a personal portfolio website showcasing my projects, skills, and experience. It is designed to be responsive and includes interactive elements.  

## Key Sections & Features
- **Navigation Bar:** Sticky navbar with links to sections and a responsive hamburger menu.  
- **About Me:** Introduction, educational background, and projects.  
- **Projects:** Featured projects with cards including title, description, image, and GitHub links.  
- **Experience:** Timeline of internships and work experiences with roles, companies, and responsibilities.  
- **Skills:** Programming languages, web & database skills, and skill levels visually represented.  
- **Chat Widget:** Simple chat interface for visitors to leave messages.  
- **Background Music:** Embedded YouTube playlist for ambiance.  
- **Footer:** Contact info.
